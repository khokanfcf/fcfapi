    Rare fares - Get All Valid Posts

    Daily alerts API filters return all valid posts from category Rare Fares based category params 607. We can also do some extra filters with below available parameters

    HTTP request

    GET
    /wp-json/fcf/v1/daily-alerts-v2/?category=607
    curl --location --request GET 'https://dev.firstclassflyer.com/wp-json/fcf/v1/daily-alerts-v2/?category=607'

    <?php

$curl = curl_init();

curl_setopt_array($curl, array(
  CURLOPT_URL => 'https://dev.firstclassflyer.com/wp-json/fcf/v1/daily-alerts-v2/?category=607',
  CURLOPT_RETURNTRANSFER => true,
  CURLOPT_ENCODING => '',
  CURLOPT_MAXREDIRS => 10,
  CURLOPT_TIMEOUT => 0,
  CURLOPT_FOLLOWLOCATION => true,
  CURLOPT_HTTP_VERSION => CURL_HTTP_VERSION_1_1,
  CURLOPT_CUSTOMREQUEST => 'GET',
));

$response = curl_exec($curl);

curl_close($curl);
echo $response;
?>

    var requestOptions = {
  method: 'GET',
  redirect: 'follow'
};

fetch("https://dev.firstclassflyer.com/wp-json/fcf/v1/daily-alerts-v2/?category=607", requestOptions)
  .then(response => response.text())
  .then(result => console.log(result))
  .catch(error => console.log('error', error));

    Request Information

    Category Value
    Method GET
    Url https://dev.firstclassflyer.com/wp-json/fcf/v1/daily-alerts-v2/?category=607

    Available Query Parameters

    Query Params Required Description
    category required Category ID of posts. For example - 607 id is for rare fares posts
    travel-to optional Ex. - 679,680 ,etc [ Require for All Destinations Filter dropdown, params values should be depends on selected values of dropdown ]
    availability optional Ex. - 2022-04,2022-05 ,etc [ Require for All Travel Months Filter dropdown, params values should be depends on selected values of dropdown ]
    airline optional Ex. - 905,357 ,etc [ Require for All Airlines Filter dropdown, params values should be depends on selected values of dropdown ]
    fares optional Ex. - 600,2250 [ Require for Fares price range filter, params value should be depends on selected range filed value (min,max) ]
    badge optional Ex. - hurry [ Require for only Rare fares posts when we want to filter by Expires Soon checkbox ]
    sort optional min/max [ Require for only when we want to sort price range as per ASC/DESC ]
